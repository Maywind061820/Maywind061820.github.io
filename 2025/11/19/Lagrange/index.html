Lagrange乘数法

Easy 两类最优化问题

无约束优化问题

min⁡f(x)\min f(\bm{x}) minf(x)

直接求导，寻找 f′(x)=0f^\prime (\bm{x}) = 0f′(x)=0 即可

有等式约束优化问题

min⁡f(x)\min f(\bm{x}) minf(x)

s.t.hi(x)=0,i=1,2,…,ns.t.h_i(\bm{x}) = 0, i = 1,2,\dots,n s.t.hi​(x)=0,i=1,2,…,n

构造拉格朗日函数

L(x,λ)=f(x)+∑i=1nλihi(x)L(\bm{x},\bm{\lambda}) = f(\bm{x}) + \sum_{i=1}^n \lambda_i h_i(\bm{x}) L(x,λ)=f(x)+i=1∑n​λi​hi​(x)

对每个变量求偏导使其为0，逐个查找。

Hard 有不等式约束优化问题

原始问题 Primal Problem

min⁡f(x)\min f(\bm{x}) minf(x)

s.t.hi(x)=0,i=1,2,…,ns.t.h_i(\bm{x}) = 0, i = 1,2,\dots,n s.t.hi​(x)=0,i=1,2,…,n

gi(x)≤0,i=1,2,…,kg_i(\bm{x}) \leq 0, i = 1,2,\dots,k gi​(x)≤0,i=1,2,…,k

构造拉格朗日函数

L(x,λ,μ)=f(x)+∑i=1nλihi(x)+∑i=1kμigi(x)L(\bm{x},\bm{\lambda},\bm{\mu}) = f(\bm{x}) + \sum_{i=1}^n \lambda_i h_i(\bm{x}) + \sum_{i=1}^k \mu_i g_i(\bm{x}) L(x,λ,μ)=f(x)+i=1∑n​λi​hi​(x)+i=1∑k​μi​gi​(x)

令

z(x)=max⁡μ≥0,λL(x,λ,μ)z(\bm{x}) = \max_{\mu \geq 0,\lambda} L(\bm{x},\bm{\lambda},\bm{\mu}) z(x)=μ≥0,λmax​L(x,λ,μ)

这个函数 z(x)z(\bm{x})z(x) 的关键性质在于它对不同 x\bm{x}x 的取值：

  • 如果 x\bm{x}x 满足所有约束（即 hi(x)=0h_i(\bm{x}) = 0hi​(x)=0 和 gi(x)≤0g_i(\bm{x}) \leq 0gi​(x)≤0）：

    • ∑λihi(x)=0\sum \lambda_i h_i(\bm{x}) = 0∑λi​hi​(x)=0，因为 hi(x)=0h_i(\bm{x}) = 0hi​(x)=0
    • ∑μigi(x)≤0\sum \mu_i g_i(\bm{x}) \leq 0∑μi​gi​(x)≤0，因为 μi≥0\mu_i \geq 0μi​≥0 和 gi(x)≤0g_i(\bm{x}) \leq 0gi​(x)≤0
    • 为了最大化 L(x,λ,μ)L(\bm{x}, \lambda, \mu)L(x,λ,μ)，选择 μi=0\mu_i = 0μi​=0
    • 因此，L(x,λ,μ)=f(x)L(\bm{x}, \lambda, \mu) = f(\bm{x})L(x,λ,μ)=f(x)，从而 z(x)=f(x)z(\bm{x}) = f(\bm{x})z(x)=f(x)

  • 如果 x\bm{x}x 违反任何约束：

    • 如果某些 hi(x)≠0h_i(\bm{x}) \neq 0hi​(x)=0，可以选择 λi\lambda_iλi​ 使 λihi(x)\lambda_i h_i(\bm{x})λi​hi​(x) 趋于无穷大
    • 如果所有 hi(x)=0h_i(\bm{x}) = 0hi​(x)=0 但某些 gi(x)>0g_i(\bm{x}) > 0gi​(x)>0，可以选择 μi\mu_iμi​ 很大正数使 μigi(x)\mu_i g_i(\bm{x})μi​gi​(x) 趋于无穷大
    • 因此，z(x)=∞z(\bm{x}) = \inftyz(x)=∞

综上，z(x)z(\bm{x})z(x) 是一个"惩罚函数"：

z(x)={f(x)如果 x 满足所有约束∞否则z(\bm{x}) = \begin{cases} f(\bm{x}) & \text{如果 } \bm{x} \text{ 满足所有约束} \\ \infty & \text{否则} \end{cases} z(x)={f(x)∞​如果 x 满足所有约束否则​

因此原问题等价于

min⁡xz(x)\min_x z(\bm{x}) xmin​z(x)

因为

  • 如果 x\bm{x}x 不满足约束，z(x)=∞z(\bm{x}) = \inftyz(x)=∞，不会被选为最小值
  • 如果 x\bm{x}x 满足约束，z(x)=f(x)z(\bm{x}) = f(\bm{x})z(x)=f(x)，所以最小化 z(x)z(\bm{x})z(x) 就是最小化 f(x)f(\bm{x})f(x) 在约束条件下

因此，通过定义 z(x)z(\bm{x})z(x)，我们将原约束优化问题转化为一个无约束优化问题。

对偶问题 Dual Problem

现在我们的 原始问题 为

min⁡xmax⁡μ≥0,λL(x,λ,μ)\min_x \max_{\mu \geq 0,\lambda} L(\bm{x},\bm{\lambda},\bm{\mu}) xmin​μ≥0,λmax​L(x,λ,μ)

交换其中的 min⁡\minmin 和 max⁡\maxmax 得到其 对偶问题 为

max⁡μ≥0,λmin⁡xL(x,λ,μ)\max_{\mu \geq 0,\lambda} \min_x L(\bm{x},\bm{\lambda},\bm{\mu}) μ≥0,λmax​xmin​L(x,λ,μ)

记

y(λ,μ)=min⁡xL(x,λ,μ)y(\bm{\lambda},\bm{\mu}) = \min_x L(\bm{x},\bm{\lambda},\bm{\mu}) y(λ,μ)=xmin​L(x,λ,μ)

称其为拉格朗日对偶函数

弱对偶性Weak Duality

记原始问题的答案为 p∗p^*p∗，而对偶问题的最优解为 d∗d^*d∗，有

d∗≤p∗d^* \leq p^* d∗≤p∗

其中 p∗−d∗p^* - d^*p∗−d∗ 称为 对偶性间隔Duality Gap

证明

对于极值点（事实上为所有满足约束条件的点） x∗\bm{x}^*x∗ 有

∑i=1nλihi(x)+∑i=1kμigi(x)≤0\sum_{i=1}^n \lambda_i h_i(\bm{x}) + \sum_{i=1}^k \mu_i g_i(\bm{x}) \leq 0 i=1∑n​λi​hi​(x)+i=1∑k​μi​gi​(x)≤0

则

L(x∗,λ,μ)=∑i=1nλihi(x)+∑i=1kμigi(x)≤f(x∗)L(\bm{x}^*,\bm{\lambda},\bm{\mu}) = \sum_{i=1}^n \lambda_i h_i(\bm{x}) + \sum_{i=1}^k \mu_i g_i(\bm{x}) \leq f(\bm{x}^*) L(x∗,λ,μ)=i=1∑n​λi​hi​(x)+i=1∑k​μi​gi​(x)≤f(x∗)

所以有

y(λ,μ)=min⁡xL(x∗,λ,μ)=∑i=1nλihi(x)+∑i=1kμigi(x)≤f(x∗)≤p∗y(\bm{\lambda},\bm{\mu}) = \min_x L(\bm{x}^*,\bm{\lambda},\bm{\mu}) = \sum_{i=1}^n \lambda_i h_i(\bm{x}) + \sum_{i=1}^k \mu_i g_i(\bm{x}) \leq f(\bm{x}^*) \leq p^* y(λ,μ)=xmin​L(x∗,λ,μ)=i=1∑n​λi​hi​(x)+i=1∑k​μi​gi​(x)≤f(x∗)≤p∗

得证。

强对偶性 Strong Duality

强对偶性为

d∗=p∗d^* = p^* d∗=p∗

在强对偶性成立的情况下，求解对偶问题同时就解决了原始问题。此时有

f(x∗)=y(λ∗,μ∗)=min⁡x[f(x)+∑i=1nλi∗hi(x)+∑i=1kμi∗gi(x)]≤f(x)+∑i=1nλi∗hi(x∗)+∑i=1kμi∗gi(x∗)≤f(x∗)\begin{aligned} f(\bm{x}^*) &= y(\bm{\lambda^*},\bm{\mu^*}) = \min_x \left[f(\bm{x}) + \sum_{i=1}^n \lambda_i^* h_i(\bm{x}) + \sum_{i=1}^k \mu_i^* g_i(\bm{x})\right] \\ &\leq f(\bm{x}) + \sum_{i=1}^n \lambda_i^* h_i(\bm{x}^*) + \sum_{i=1}^k \mu_i^* g_i(\bm{x}^*) \leq f(\bm{x}^*) \end{aligned} f(x∗)​=y(λ∗,μ∗)=xmin​[f(x)+i=1∑n​λi∗​hi​(x)+i=1∑k​μi∗​gi​(x)]≤f(x)+i=1∑n​λi∗​hi​(x∗)+i=1∑k​μi∗​gi​(x∗)≤f(x∗)​

两头相等，因此不等号可变为等号。

有第一个不等号可知 x∗\bm{x}^*x∗ 为 L(x,λ∗,μ∗)L(\bm{x},\lambda^*,\mu^*)L(x,λ∗,μ∗) 的一个极值点，故 ∇L(x∗,λ∗,μ∗)=0\nabla L(\bm{x}^*,\lambda^*,\mu^*)=0∇L(x∗,λ∗,μ∗)=0

此外有 μi∗gi(x∗)=0\mu_i^* g_i(\bm{x}^*) = 0μi∗​gi​(x∗)=0，综合原始问题的条件可得 KKT(Karush-Kuhn-Tucker)条件

hi(x∗)=0,i=1,2,…,ngi(x∗)≤0,i=1,2,…,kPrimal Feasibility (原问题可行性)μi∗≥0,i=1,2,…,kDual Feasibility (对偶可行性)μi∗gi(x∗)=0Complementary Slackness (互补松弛条件)∇xL(x∗,λ∗,μ∗)=0Stationarity (平稳性)\begin{aligned} &h_i(\bm{x}^*) = 0, i = 1,2,\dots,n \\ &g_i(\bm{x}^*) \leq 0, i = 1,2,\dots,k \quad &\text{Primal Feasibility (原问题可行性)}\\ &\mu_i^* \geq 0,i = 1,2,\dots,k \quad &\text{Dual Feasibility (对偶可行性)}\\ &\mu_i^* g_i(\bm{x}^*) = 0 \quad &\text{Complementary Slackness (互补松弛条件)}\\ &\nabla_x L(\bm{x}^*,\bm{\lambda}^*,\bm{\mu}^*) = 0 \quad &\text{Stationarity (平稳性)} \end{aligned} ​hi​(x∗)=0,i=1,2,…,ngi​(x∗)≤0,i=1,2,…,kμi∗​≥0,i=1,2,…,kμi∗​gi​(x∗)=0∇x​L(x∗,λ∗,μ∗)=0​Primal Feasibility (原问题可行性)Dual Feasibility (对偶可行性)Complementary Slackness (互补松弛条件)Stationarity (平稳性)​
