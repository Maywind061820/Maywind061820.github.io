Cox比例风险模型的数学原理深度解析

1. 模型的定义与动机

Cox模型的核心是风险函数。在生存分析中，风险函数 λ(t)（或称 h(t)）定义为：

λ(t) = lim (Δt→0) [ P(t ≤ T < t+Δt | T ≥ t) / Δt ]

它表示一个个体在存活到时间 t 的前提下，在接下来极短的单位时间内发生事件的瞬时概率。

David Cox在1972年提出的关键思想是将风险函数分解为两个独立的部分：

λ(t|X) = λ₀(t) * exp(βᵀX)

其中：

  • λ₀(t)：基准风险函数，是任意形式的、只与时间 t 有关的函数。它代表了当所有协变量 X 都为0时的风险。
  • exp(βᵀX)：风险比，是一个与时间无关的、由协变量 X 线性组合的指数函数。β 是待估计的系数向量。

这种分解的动机在于：

  • 我们通常关心的是协变量 X（如治疗方案、年龄）对风险的相对影响，而非风险的绝对水平。
  • 通过这种形式，我们无需对复杂且通常未知的 λ₀(t) 做出任何参数假设（如它是指数分布或威布尔分布），从而使模型非常灵活，被称为半参数模型。

2. 偏似然函数的推导：数学起源

Cox的第二个关键贡献是提出了偏似然函数，它允许我们在不估计 λ₀(t) 的情况下估计系数 β。其推导过程如下：

a) 基本设定
假设我们有 n 个个体，观察到 k 个互不相同的死亡时间 t₁ < t₂ < ... < tₖ（为简化，先假设无同时发生事件）。设 R(tᵢ) 为在时间 tᵢ 之前的风险集，即所有在 tᵢ 时刻仍存活且处于研究中的个体集合。

b) 条件概率思想
在某个死亡时间 tᵢ，假设是第 i 个个体死亡。给定风险集 R(tᵢ) 中恰好有一个个体在 tᵢ 死亡，那么这个个体是 i（而非风险集中的其他任何人）的条件概率是多少？

这个概率可以写为：
Lᵢ = P(个体i死于tᵢ | R(tᵢ)中有一人死亡) = λ(tᵢ|Xᵢ) / Σⱼ∈R(tᵢ) λ(tᵢ|Xⱼ)

c) 代入Cox模型
将 λ(t|X) = λ₀(t) * exp(βᵀX) 代入上式：

Lᵢ = [λ₀(tᵢ) * exp(βᵀXᵢ)] / Σⱼ∈R(tᵢ) [λ₀(tᵢ) * exp(βᵀXⱼ)]

注意到 λ₀(tᵢ) 是公共项，可以在分子和分母中约去：

Lᵢ = exp(βᵀXᵢ) / Σⱼ∈R(tᵢ) exp(βᵀXⱼ)

这是一个极其重要的结果。它意味着，在每一个死亡时间点，给定风险集中发生一个死亡事件，该事件落到特定个体 i 身上的条件概率，完全取决于该个体的协变量 Xᵢ 与风险集中所有个体协变量的比较，而与未知的、讨厌的基准风险函数 λ₀(t) 无关。

d) 构建偏似然函数
假设不同死亡时间之间是独立的，那么观察到整个数据中所有死亡事件按实际顺序发生的联合概率，就是所有 Lᵢ 的乘积：

PL(β) = ∏ᵢ₌₁ᵏ Lᵢ = ∏ᵢ₌₁ᵏ [ exp(βᵀXᵢ) / Σⱼ∈R(tᵢ) exp(βᵀXⱼ) ]

这个 PL(β) 就是著名的Cox偏似然函数。它“偏”在因为它不是基于完整数据的似然函数，而是基于事件发生顺序所提供的信息构建的条件似然。

3. 参数估计：如何计算系数 β

我们的目标是找到一组系数 β̂，使得这个偏似然函数 PL(β) 的值达到最大。这称为最大偏似然估计。

a) 对数偏似然函数
在实践中最常用的是最大化对数偏似然函数，因为它能将连乘转化为连加，更易于处理：

ll(β) = log[PL(β)] = Σᵢ₌₁ᵏ { βᵀXᵢ - log[ Σⱼ∈R(tᵢ) exp(βᵀXⱼ) ] }

b) 得分函数与信息矩阵
为了找到最大值点，我们求 ll(β) 关于 β 的一阶偏导数（即得分函数），并令其为零：

U(β) = ∂ll(β)/∂β = Σᵢ₌₁ᵏ { Xᵢ - [ Σⱼ∈R(tᵢ) Xⱼ exp(βᵀXⱼ) ] / [ Σⱼ∈R(tᵢ) exp(βᵀXⱼ) ] } = 0

这个方程 U(β) = 0 通常没有解析解（闭式解）。

c) 数值优化算法
因此，我们需要采用迭代数值算法来求解 β̂。最常用的是牛顿-拉弗森算法。

  1. 初始化：选择一个初始值 β⁽⁰⁾（例如，全零向量）。
  2. 迭代更新： β⁽ᵐ⁺¹⁾ = β⁽ᵐ⁾ - [I(β⁽ᵐ⁾)]⁻¹ U(β⁽ᵐ⁾)
    • 其中 I(β) 是观测信息矩阵，即对数偏似然函数的二阶导数矩阵（海森矩阵）的负值： I(β) = -∂²ll(β)/∂β∂βᵀ。它衡量了似然函数曲率的陡峭程度。
  3. 收敛判断：当 β 的更新值变化非常小（例如，小于一个预设的阈值 ε），或者得分函数 U(β) 足够接近零时，算法停止。此时的 β⁽ᵐ⁾ 即为最大偏似然估计量 β̂。

d) 处理结数据
当多个事件在同一时间发生时，就产生了“结”。此时上述偏似然需要修正。常用方法有：

  • Breslow近似： Lᵢ = [ ∏_{l∈D(tᵢ)} exp(βᵀX_l) ] / [ Σⱼ∈R(tᵢ) exp(βᵀXⱼ) ]^{dᵢ}
    (其中 D(tᵢ) 是在 tᵢ 时刻发生事件的个体集合，dᵢ 是该集合的大小)
  • Efron近似： 比Breslow更精确，是默认的常用方法。
  • 精确偏似然： 计算量最大，但最精确。

4. 结果解读与统计推断

得到 β̂ 后：

  • 风险比： HR = exp(β̂)。这是核心解释指标。例如，对于一个二分类变量（如用药=1 vs 用药=0），HR = 2 意味着用药组的风险是对照组的2倍。
  • 系数显著性：基于最大似然理论，β̂ 近似服从正态分布。我们可以构造Wald检验统计量： Z = β̂ / se(β̂)，其平方近似为卡方分布，用于计算p值。
  • 置信区间： β̂ 的95%置信区间为 β̂ ± 1.96 * se(β̂)。风险比 HR 的95%置信区间则为 exp(β̂ ± 1.96 * se(β̂))。

总结

Cox回归的数学原理可以概括为：

  1. 模型设定：通过 λ(t|X) = λ₀(t) * exp(βᵀX) 将风险分解为时间相关和协变量相关两部分。
  2. 似然构造：利用条件概率思想，在每一个事件发生时，比较事件个体与风险集中其他个体，巧妙地消去了未知的 λ₀(t)，构建出偏似然函数。
  3. 参数估计：通过牛顿-拉弗森等数值优化算法，最大化对数偏似然函数，求解出系数估计值 β̂。
  4. 统计推断：基于 β̂ 的渐近分布性质，计算风险比 HR、p值和置信区间，从而对协变量的影响做出统计结论。

这个过程完美地体现了统计建模的智慧：通过合理的假设和巧妙的构造，绕过无法估计的“ nuisance parameter ” (λ₀(t))，直接对我们关心的参数 (β) 进行有效的估计和推断。
