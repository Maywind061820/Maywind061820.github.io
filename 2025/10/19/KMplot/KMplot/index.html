1. 数学模型与原理推导

1.1 基本定义与数据结构

设 TTT 为一非负随机变量，表示从起始事件到终点事件发生的时间，称为生存时间。其累积分布函数为 F(t)=P(T≤t)F(t) = P(T \leq t)F(t)=P(T≤t)，我们定义其互补函数，即生存函数 S(t)S(t)S(t) 为：

S(t)=P(T>t)=1−F(t)S(t) = P(T > t) = 1 - F(t) S(t)=P(T>t)=1−F(t)

S(t)S(t)S(t) 表示个体在时间 ttt 之后仍未发生终点事件的概率。

在实际研究中，由于研究周期终止或个体失访，我们常无法观测到所有个体的确切生存时间 TTT，此即删失现象。假设存在一个与 TTT 独立的非负随机删失时间 CCC。我们实际观测到的数据为：

  • 观测时间： Xi=min⁡(Ti,Ci)X_i = \min(T_i, C_i)Xi​=min(Ti​,Ci​)
  • 事件指示变量： δi=I(Ti≤Ci)\delta_i = I(T_i \leq C_i)δi​=I(Ti​≤Ci​)

数据集由 nnn 个独立同分布的观测对 (Xi,δi)(X_i, \delta_i)(Xi​,δi​) 构成。

1.2 Kaplan-Meier估计量推导

Kaplan-Meier估计量采用乘积极限法来估计生存函数 S(t)S(t)S(t)。其核心思想是将生存概率的估计分解为在各个有序事件时间点上的条件概率的连续乘积。

设观测数据中存在 kkk 个互异的事件发生时间点，将其排序为 t1<t2<⋯<tkt_1 < t_2 < \dots < t_kt1​<t2​<⋯<tk​。

对于任意时间点 ttt，定义：

  • 风险集 R(t)={i:Xi≥t}R(t) = \{ i: X_i \geq t \}R(t)={i:Xi​≥t}，即在时间 ttt 之前尚未发生事件或删失，仍处于"风险"中的个体集合。
  • 在事件时间点 tjt_jtj​，定义：
    • nj=∣R(tj)∣n_j = |R(t_j)|nj​=∣R(tj​)∣ 为在 tjt_jtj​ 时刻处于风险中的个体数。
    • djd_jdj​ 为在 tjt_jtj​ 时刻发生终点事件的个体数。

则在时间 tjt_jtj​ 及其附近区间内，在已知个体存活至 tjt_jtj​ 的条件下，其在此区间内死亡的条件概率的矩估计为 dj/njd_j / n_jdj​/nj​。相应地，其存活过此区间的条件概率为 1−dj/nj1 - d_j / n_j1−dj​/nj​。

2. 生存曲线解读指南

2.1 图形化特征分析

Kaplan-Meier生存曲线图是 S^(t)\hat{S}(t)S^(t) 相对于时间 ttt 的图形化表示。

  • 曲线形态：曲线为从 S^(0)=1\hat{S}(0) = 1S^(0)=1 开始的、单调非增的右连续阶梯函数。
  • 中位生存时间：定义为满足 S^(t)=0.5\hat{S}(t) = 0.5S^(t)=0.5 的时间 ttt。
  • 删失标记：图中常以"+"或短竖线标记删失观测发生的位置。

2.2 多组曲线比较与假设检验

视觉上比较不同组别的KM曲线是初步分析。然而，视觉比较主观，需辅以正式的统计检验。最常用的是Log-Rank检验。

  • 原假设 H0H_0H0​：所有组别的生存函数在全时间域上相等。
  • 备择假设 H1H_1H1​：至少有一组在全时间域上的生存函数与其他组不同。

2.3 风险比

作为Log-Rank检验的补充，风险比 提供了一个效应大小的量化指标。在满足比例风险假定的前提下，可通过Cox比例风险模型得到风险比 HRHRHR 的估计值：

HR=h1(t)h2(t)HR = \frac{h_1(t)}{h_2(t)} HR=h2​(t)h1​(t)​

其中 h(t)h(t)h(t) 是风险函数。HR>1HR > 1HR>1 表示第一组相对于第二组发生事件的风险更高。
